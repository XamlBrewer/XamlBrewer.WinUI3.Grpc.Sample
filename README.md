# XamlBrewer.WinUI3.Grpc.Sample

Demonstrates building a gRPC client WinUI 3 Desktop application, against an ASP.NET Core gRPC server:
* calling and streaming,
* deadlines,
* logging,
* client-side load balancing,
* health monitoring.
